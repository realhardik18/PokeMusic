Welcome to PokeMusic

PokeMusic uses the PokeAPI and the SpotifyAPI to answer life's most sought after question

What type of songs would a pokemon hear?

The way it works is that PokeMusic first gets the stats of a particular pokemon

Now these stats are fed to an algorithim which then returns the target tempo,loudness etc of songs

Then it searches for songs with the help of the spotify api

And get a list of 10 songs the pokemon would listen to!

Use this buttons to get the songs a pokemon would listen to!
